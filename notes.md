# inicializar o node no projeto

npm init es6 -y (usar o command prompt)

# instalar dependências

npm install express

# executar o server

node server.js
node --watch server.js  (reinicia o servidor automaticamente em caso de alterações)

## Depois de configurar o sript

npm run dev
